2. Overview of Data Processing

This website (return2beta.com) is a non-commercial fan project for a Minecraft Beta 1.7.3 server. We process personal data only to the extent necessary to provide the website and its features.

3. Hosting

The static website is hosted via GitHub Pages (GitHub Inc., USA). When you access the website, technically necessary connection data (including your IP address) is processed by GitHub. GitHub’s privacy policy: GitHub Privacy Statement.

The chat server and WebSocket backend run on a STRATO VPS (STRATO AG, Germany). Connection data (IP address) is processed to maintain the WebSocket connection. IP addresses are recorded in server and connection logs as a technical necessity.

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in providing a reliable and secure website).

4. Data We Collect

4.1 Chat (WebSocket)

When you use the chat feature, the following data is processed:

  • Username – a display name you freely choose (no registration required)
  • Chat messages – the text content you send
  • IP address – transmitted via the WebSocket connection

Chat messages are stored temporarily in server memory to provide message history to new visitors and are cleared when the server restarts. In addition, messages are forwarded to a Discord server (Discord Inc., USA) via a chat bridge, where they are processed and stored according to Discord’s own policies. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in providing a community chat).

4.2 Local Storage (Browser)

We use your browser’s localStorage to remember your preferences. The following keys are stored locally on your device only and are never transmitted to our servers unless noted:

  • r2b_chat_username – your chosen chat display name (sent to server when using chat)
  • r2b_map_collapsed – map panel UI state
  • r2b_wiki_dark – wiki dark mode preference
  • r2b_legal_lang – language preference for legal pages
  • Game high score keys – locally cached personal best scores
  • r2b_starred_plugins – bookmarked plugins on the plugins page

Legal basis: Art. 6(1)(f) GDPR / § 25(2) TDDDG (strictly necessary for the requested service functionality).

4.3 Game High Scores

Site-wide high scores for the browser games are stored on our server in a JSON file. Only the highest score per game is stored – no usernames or IP addresses are associated with scores. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in providing game functionality).

4.4 Visitor Counter

We display a live visitor count showing how many users are currently on the site. To determine the number of active connections, technically necessary connection data is briefly processed; only the numerical count itself is stored persistently in memory. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in displaying site activity).

5. Third-Party Services

5.1 Fonts

All fonts used on this website are self-hosted on our own servers. No external font CDN is used, and no data is transmitted to third parties for font loading.

5.2 Embedded Content (iframes)

Some pages embed third-party content via iframes:

  • Dynmap (map.return2beta.com) – our own server map
  • Mine Blocks (mineblocks.com) – third-party game
  • Mine Blocks 2 (zanzlanz.com) – third-party game
  • Wayback Machine (web.archive.org) – archived Minecraft Wiki

Third-party iframes are loaded only after you explicitly click a consent placeholder. No data is transmitted to these third parties until you choose to load the content. These third-party sites may then process your data according to their own privacy policies. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in embedding relevant content).

5.3 CORS Proxy

To check if the Dynmap server is online, we may send requests through corsproxy.io (operated by Hiroki Zenigami) as a fallback. Your IP address may be transmitted to this service. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in verifying server status).

6. Cookies

This website does not set any cookies itself. Third-party iframes may set their own cookies – please refer to the respective third-party privacy policies.

7. Data Retention

  • Chat messages: stored temporarily in server memory, cleared on restart; additionally stored on Discord according to their retention policies
  • Nginx server logs (containing IP addresses): rotated daily, deleted after 14 days
  • Chat application log: does not contain IP addresses or personal data
  • localStorage data: persists in your browser until you clear it
  • High scores: stored indefinitely (no personal data)
