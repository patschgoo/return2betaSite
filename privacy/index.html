2. Overview of Data Processing

This website (return2beta.com) is a non-commercial fan project for a Minecraft Beta 1.7.3 server. We process personal data only to the extent necessary to provide the website and its features.

3. Hosting

The static website is hosted via GitHub Pages (GitHub Inc., USA). When you visit the site, GitHub may collect your IP address in server logs. GitHub’s privacy policy: GitHub Privacy Statement.

The chat server and WebSocket backend run on a STRATO VPS (STRATO AG, Germany). Connection data (IP address) is processed to maintain the WebSocket connection. Server logs may temporarily contain IP addresses.

4. Data We Collect

4.1 Chat (WebSocket)

When you use the chat feature, the following data is processed:

  • Username – a display name you freely choose (no registration required)
  • Chat messages – the text content you send
  • IP address – transmitted via the WebSocket connection

Chat messages are stored in memory on the server to provide message history to new visitors. They are not permanently stored on disk and are cleared when the server restarts. Legal basis: Art. 6(1)(f) GDPR (legitimate interest in providing a community chat).

4.2 Local Storage (Browser)

We use your browser’s localStorage to remember your preferences. The following keys are stored locally on your device only and are never transmitted to our servers unless noted:

  • r2b_chat_username – your chosen chat display name (sent to server when using chat)
  • r2b_map_collapsed – map panel UI state
  • r2b_wiki_dark – wiki dark mode preference
  • r2b_legal_lang – language preference for legal pages
  • Game high score keys – locally cached personal best scores

Legal basis: Art. 6(1)(f) GDPR / § 25(2) TTDSG (strictly necessary for the requested service functionality).

4.3 Game High Scores

Site-wide high scores for the browser games are stored on our server in a JSON file. Only the highest score per game is stored – no usernames or IP addresses are associated with scores.

4.4 Visitor Counter

We display a live visitor count showing how many users are currently on the site. This is determined by counting active WebSocket connections. No personal data is stored for this purpose – only a numerical count is maintained in memory.

5. Third-Party Services

5.1 jsDelivr CDN

We load a font file from jsDelivr (Prospect One, Poland). When your browser requests this file, your IP address is transmitted to jsDelivr’s servers. Privacy policy: jsDelivr Privacy Policy.

5.2 Embedded Content (iframes)

Some pages embed third-party content via iframes:

  • Dynmap (map.return2beta.com) – our own server map
  • Mine Blocks (mineblocks.com) – third-party game
  • Mine Blocks 2 (zanzlanz.com) – third-party game
  • Wayback Machine (web.archive.org) – archived Minecraft Wiki

These third-party sites may process your data according to their own privacy policies.

5.3 CORS Proxy

To check if the Dynmap server is online, we may send requests through corsproxy.io as a fallback. Your IP address may be transmitted to this service.

6. Cookies

This website does not set any cookies itself. Third-party iframes may set their own cookies – please refer to the respective third-party privacy policies.

7. Data Retention

  • Chat messages: kept in server memory only, cleared on restart
  • Server logs: may contain IP addresses, rotated regularly
  • localStorage data: persists in your browser until you clear it
  • High scores: stored indefinitely (no personal data)

4. Welche Daten wir erheben

4.1 Chat (WebSocket)

Bei Nutzung der Chat-Funktion werden folgende Daten verarbeitet:

  • Chat-Nachrichten – der Textinhalt Ihrer Nachrichten

Chat-Nachrichten werden im Arbeitsspeicher des Servers gespeichert, um neuen Besuchern den Nachrichtenverlauf bereitzustellen. Sie werden nicht dauerhaft auf der Festplatte gespeichert und bei einem Neustart des Servers gelöscht. 4.2 Lokaler Speicher (Browser)

Wir nutzen den localStorage Ihres Browsers, um Ihre Einstellungen zu speichern. Folgende Schlüssel werden ausschließlich lokal auf Ihrem Gerät gespeichert und nur wie angegeben an unsere Server übermittelt:

Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO / § 25 Abs. 2 TTDSG (technisch notwendig für die angeforderte Dienstfunktionalität).

4.3 Spiel-Highscores

Seitenweite Highscores für die Browserspiele werden auf unserem Server in einer JSON-Datei gespeichert. Es wird nur der höchste Punktestand pro Spiel gespeichert – keine Benutzernamen oder IP-Adressen werden mit den Punkteständen verknüpft.

4.4 Besucherzähler

Wir zeigen einen Live-Besucherzähler an, der die Anzahl der aktuell auf der Website befindlichen Nutzer angibt. Dies wird durch Zählen der aktiven WebSocket-Verbindungen ermittelt. Es werden keine personenbezogenen Daten gespeichert – nur ein numerischer Zähler wird im Arbeitsspeicher gehalten.
