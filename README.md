# Return2Beta Website

The official website for **Return2Beta**, a Minecraft Beta 1.7.3 community server running [Poseidon](https://github.com/retromcorg/Poseidon) (a Bukkit b1.7.3 fork).

**Live at:** [return2beta.com](https://return2beta.com)

## Pages

- **Home** – Server overview with a live chat widget
- **Map** – Dynmap world viewer
- **Plugins** – Browsable plugin list with readme files and server configuration
- **Wiki** – Embedded archived Minecraft Wiki (with dark mode toggle)
- **Chat** – Full web chat bridged to Discord
- **How to Join** – Step-by-step launcher guides, server IPs, and recommended mods

## Tech

- Static site hosted on GitHub Pages
- Chat backend: Node.js + WebSocket server bridged to Discord via discord.js
- Minecraft font via [IdreesInc/Minecraft-Font](https://github.com/IdreesInc/Minecraft-Font)
