§6=== Chat Commands ===
§e/msg <player> <message> §f- Private message
§e/r <message> §f- Reply to last PM
§e/me <action> §f- Action text (*you do something*)
§e/nick <name> §f- Set nickname (shown as ~name)
§e/afk §f- Toggle away-from-keyboard
§e/list §f- See who's online
§e/ignore <player> §f- Block a player's messages
§e/helpop <message> §f- Send help request to staff
§e/compass §f- Direction you're facing
§e/depth §f- Your Y-level (height)
