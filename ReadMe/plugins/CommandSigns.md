# CommandSigns

## What It Does
CommandSigns allows special signs placed around the world to execute commands when right-clicked. Staff create these signs, and players can use them.

## How It Affects You
Members can use all types of command signs placed around the server. These signs might teleport you, give you items, or trigger other actions.

## How to Use
- Simply **right-click** a command sign to activate it.
- The sign will execute whatever command(s) staff have assigned to it.

## Notes
- Command signs are placed and configured by staff — you cannot create your own.
- Some signs may cost money to use (via iConomy integration).
- Look for signs with special formatting or instructions around spawn and public areas.
