# Stackable

## What It Does
Stackable modifies the maximum stack sizes for various items, allowing some items to stack higher or lower than vanilla defaults.

## How It Affects You
Some items stack differently than you might expect from vanilla Minecraft:
- **Tools and weapons** stack to 1 (same as vanilla).
- **Armor pieces** stack to 1 (same as vanilla).
- **Food items** stack to 8.
- **Vehicles** (boats, minecarts) stack to 8.
- **Signs** stack to 16.
- **Doors** stack to 1.
- **Buckets** stack to 16.
- **Saddles** stack to 4.
- **Snowballs and eggs** stack to 16.

## Notes
- No commands — stack sizes are applied automatically.
- This mainly affects how you organize your inventory.
