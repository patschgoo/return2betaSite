# OtherDrops

## What It Does
OtherDrops allows customization of what items drop when blocks are broken or mobs are killed. It can add, modify, or replace drop tables.

## How It Affects You
On this server, OtherDrops is installed but has **no custom drops configured**. All blocks and mobs drop their default vanilla items.

## Current Drop List
No custom drops are configured. Everything drops as normal.

## Notes
- No commands for regular players.
- If custom drops are added in the future, you'll notice different items dropping from blocks or mobs.
