# Reporter

## What It Does
Reporter lets you file reports about rule-breaking players or issues. Staff members are notified and can review the reports.

## How It Affects You
Members can file unlimited reports (Visitors are limited). This is the primary way to report griefing, stealing, or other issues to the staff.

## Commands
| Command | Description |
|---------|-------------|
| `/report <message>` | Submit a report describing the issue. |
| `/report help` | View Reporter help. |

## Tips
- Include as many details as possible: who, what, where, when.
- Reports are stored in a database and reviewed by staff.
- Staff are notified when reports are submitted.
- Use this instead of arguing with rule-breakers in chat.
