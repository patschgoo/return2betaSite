# Essentials

## What It Does
Essentials is the core utility plugin on the server. It provides teleportation, homes, economy, mail, kits, and many other everyday commands for players.

## How It Affects You
- You start with $0. Earn money through quests, selling items, or receiving payments from other players.
- The currency symbol is `$`.
- Nicknames are prefixed with `~` so others know it's a nickname.
- Auto-AFK kicks in after 5 minutes of inactivity.
- Teleport commands have no cooldown or delay on this server.
- Fire spread and lava fire spread are disabled for safety.
- PvP is enabled.
- The chat is global (no distance limit).

## Server Rules
1. No griefing or stealing.
2. No cheating, hacking, or exploiting bugs.
3. No spamming, advertising, or harassing other players.
4. No killing new or AFK players. Fair PvP only.
5. Respect staff decisions. Use `/report` for issues.
6. No lava/water dumping or TNT abuse.
7. Protect your base! Use protection stones (see PreciousStones).
8. Lock your chests with `/cprivate` or by using a lever on them.
9. Abusing PreciousStones blocks is a bannable offense.
10. Use common sense. If it feels wrong, don't do it.

## Commands
| Command | Description |
|---------|-------------|
| `/home` | Teleport to your home. |
| `/sethome` | Set your current location as home. |
| `/delhome` | Delete your home location. |
| `/spawn` | Teleport to the server spawn. |
| `/back` | Return to your last location (works after death too). |
| `/tpa <player>` | Request to teleport to another player. |
| `/tpahere <player>` | Request another player to teleport to you. |
| `/tpaccept` | Accept a teleport request. |
| `/tpdeny` | Deny a teleport request. |
| `/balance` | Check your current money balance. |
| `/pay <player> <amount>` | Send money to another player. |
| `/sell <item>` | Sell an item to the server. |
| `/worth [item]` | Check the sell value of an item. |
| `/buy` | Use buy signs in the world. |
| `/kit` | View and claim available kits. |
| `/mail send <player> <msg>` | Send mail to a player. |
| `/mail read` | Read your mail. |
| `/mail clear` | Clear your mail. |
| `/me <message>` | Send an action message. |
| `/msg <player> <message>` | Send a private message. |
| `/r <message>` | Reply to the last private message. |
| `/nick <name>` | Set your nickname (prefixed with ~). |
| `/afk` | Toggle AFK status. |
| `/list` | See who is online. |
| `/seen <player>` | Check when a player was last online. |
| `/time` | Check the current in-game time. |
| `/depth` | Check your current Y-coordinate depth. |
| `/getpos` | Show your current coordinates. |
| `/compass` | Show the direction you're facing. |
| `/motd` | View the message of the day. |
| `/rules` | View the server rules. |
| `/helpop <message>` | Send a help request to online staff. |
| `/ignore <player>` | Ignore a player's messages. |
| `/suicide` | Kill yourself (respawn at spawn). |
| `/clearinventory` | Clear your own inventory. |
| `/warp [name]` | Teleport to a server warp. |
| `/warp list` | List available warps. |
| `/help` | View available commands. |

## Essentials Signs
You can use special signs placed around the world:
- **[Buy]** / **[Sell]** – Trade items for money.
- **[Free]** – Get free items.
- **[Disposal]** – Dispose of unwanted items.
- **[Heal]** – Heal yourself.
- **[Warp]** – Teleport to a warp.
- **[Mail]** – Check your mail.
- **[Balance]** – Check your balance.
- **[Time]** – Check the time.
- **[Weather]** – Check the weather.
- **[Protection]** – Protect items.
- **[Trade]** – Trade with other players.
