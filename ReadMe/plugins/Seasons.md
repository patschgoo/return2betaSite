# Seasons

## What It Does
Seasons adds a seasonal cycle to the server with different weather patterns. The world alternates between **Winter** and **Summer**, each with unique weather probabilities.

## How It Affects You
Members can check the current season. Weather changes based on the season, and special day announcements appear.

## Season Details

### Winter
- **Duration:** 15 days
- **Weather:** 50% rain, 10% thunder, 20% sun
- **Snow/Ice:** Regenerates during Winter.
- **Special Days:**
  - Day 0 — First Day of Winter
  - Day 7 — Mid Winter
  - Day 14 — Last Day of Winter

### Summer
- **Duration:** 20 days
- **Weather:** 3% rain, 1% thunder, 90% sun
- **Special Days:**
  - Day 0 — First Day of Summer
  - Day 19 — Last Day of Summer

## Commands
| Command | Description |
|---------|-------------|
| `/seasons list` | Check the current season and how many days remain. |

## Notes
- Weather changes occur every 3 in-game days within each season.
- Special day announcements appear in chat (e.g., "It's First Day of Winter").
- Season change announcements are broadcast: "Seasons changes to [season]."
