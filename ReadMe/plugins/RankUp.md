# RankUp

## What It Does
RankUp allows Visitors to rank up to Member by using a special sign.

## How It Affects You
If you're a Visitor, you can rank up to Member by finding and clicking the **RankUp sign** on the server. This unlocks all Member permissions and commands.

## How It Works
- **Starting rank:** Visitor
- **Ending rank:** Member
- **Method:** Sign-based — find the RankUp sign and click it.
- Command-based and word-based rankup are disabled.

## Notes
- Once you rank up to Member, you gain access to homes, economy, clans, protection, and many more features.
- The RankUp sign is typically located near spawn.
- You only need to rank up once — it's permanent.
