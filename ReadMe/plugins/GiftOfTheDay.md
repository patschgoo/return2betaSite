# GiftOfTheDay

## What It Does
GiftOfTheDay gives every player a free random gift once per day. Gifts range from common supplies to rare items.

## How It Affects You
Each day you can claim a free gift. The rarity is randomly determined.

## Gift Tiers

### Common (~57% chance)
| Item | Amount |
|------|--------|
| Bread | 1–5 |
| Coal | 1–7 |
| Arrows | 1–15 |
| Bone | 1–5 |
| String | 1–4 |

### Uncommon (~24% chance)
| Item | Amount |
|------|--------|
| Iron Ingot | 2–5 |
| Redstone | 4–10 |
| Leather | 3–8 |
| Cookie | 5–13 |
| Gunpowder | 3–8 |

### Rare (~15% chance)
| Item | Amount |
|------|--------|
| Gold Ingot | 2–5 |
| Glowstone Dust | 2–5 |
| Apple | 3–8 |

### Epic (~3% chance)
| Item | Amount |
|------|--------|
| Diamond | 1–2 |
| Golden Apple | 1 |

## Commands
| Command | Description |
|---------|-------------|
| `/gift` | Claim your daily gift. |

## Notes
- You can only claim one gift per day.
- Gifts reset daily.
