# Showcase

## What It Does
Showcase lets players display and sell items by placing them visually on blocks in the world. It acts as a visual shop system.

## How It Affects You
You can create, view, and purchase items from showcases placed around the server. All showcase types are **free to create**.

## Showcase Types

| Type | Description |
|------|-------------|
| **Basic** | Display an item on a block for all to see. |
| **Finite** | Sell a limited stock of items for money. Once stock runs out, the showcase is empty. |
| **Exchange** | Trade items with other players (item for item swap). |
| **Infinite** | Unlimited items for money (admin shops only). |
| **Tutorial** | Display an item with informational text. |

## How to Create a Showcase

1. **Hold the item** you want to showcase in your hand.
2. **Right-click a block** where you want the showcase to appear.
3. Type `/showcase create` to begin the creation assistant.
4. The assistant will guide you through the options:
   - Choose the **type** (basic, finite, exchange, tutorial).
   - Set the **price** (for finite type) or **exchange item** (for exchange type).
   - Confirm to create the showcase.

### Quick Commands
| Command | Description |
|---------|-------------|
| `/showcase create` | Start the showcase creation assistant. |
| `/showcase remove` | Remove a showcase you own (look at it first). |
| `/showcase get` | Pick up an item from a showcase. |

## Features
- Showcases are **protected** — other players cannot break them.
- Items are auto-saved every 60 seconds.
- Empty showcases stay in place (not auto-removed).

## Notes
- Walk up to a showcase (an item floating above a block) and interact with it to view details or buy.
- All types are free to create — no cost to set up a shop.
