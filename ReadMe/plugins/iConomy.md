# iConomy

## What It Does
iConomy is the server's economy system. It manages player money (Dollars), balances, and transactions. Other plugins use iConomy to charge for services like chest protection, teleportation, and clan creation.

## How It Affects You
- Every new player starts with **$30.00**.
- Currency is displayed as **Dollars** (e.g., "1.23 Dollars").
- Your money on hand can be spent on various plugin features across the server.
- Banking is disabled on this server.

## Commands
| Command | Description |
|---------|-------------|
| `/money` | Check your current balance. |
| `/money pay <player> <amount>` | Send money to another player. |
| `/money rank` | See your wealth ranking. |
| `/money top` | See the richest players on the server. |
