# QuestPlus

## What It Does
QuestPlus provides a dynamic quest system where you can take on randomly generated quests to earn money and item rewards. Quests are offered by NPCs or through commands, and include collecting items, mining, killing mobs, and more.

## How It Affects You
Members can accept and complete quests. You can have up to **3 active quests** at a time. Quests have varying difficulty tiers and reward scaling.

## Quest Types
| Type | Description |
|------|-------------|
| **Collect** | Gather a specific number of items (e.g., 16–64 Cobblestone, 8–24 Coal). |
| **Kill** | Kill a number of hostile mobs (Zombies, Spiders, Skeletons, Creepers, etc.). |
| **Kill Animal** | Kill a number of passive mobs (Chickens, Cows, Pigs, Sheep, etc.). |
| **Kill Giant** | Hunt down Giants — extremely dangerous (200 HP, fire attacks). Multi-objective only. |
| **Kill Human** | Kill NPC Humans that spawn in the wild. Multi-objective only. |
| **Mine** | Mine specific ore or block types (Coal Ore, Iron Ore, Diamond Ore, etc.). |
| **Place** | Place a certain number of blocks (Dirt, Cobblestone, Planks, etc.). |
| **Kill Player** | Defeat other players in PvP (1–2 players). |
| **Multi-Objective** | A quest combining multiple objectives (25% chance to appear). |

## Rewards
- **Money:** Ranges from $5 to $250 depending on quest difficulty.
- **Items:** Possible item rewards include String, Bone, Iron Ingot, Gold Ingot, Redstone, Glowstone Dust, Slimeball, Diamond, and Golden Apple (rarer items at higher difficulty tiers).

## Difficulty Tiers
Quests come in 5 difficulty tiers. Higher tiers offer harder objectives but better rewards. Your quest difficulty range scales with your quest level (max level 50).

## Commands
| Command | Description |
|---------|-------------|
| `/quest` | View your active quests. |
| `/quest accept` | Accept an offered quest. |
| `/quest decline` | Decline an offered quest. |
| `/quest abandon` | Abandon an active quest. |
| `/quest progress` | Check progress on your active quests. |
| `/quest leaderboard` | View the quest leaderboard (resets weekly). |

## Tips
- Quests are offered every 5 minutes if you have space for more.
- Global quests appear every 60 minutes for all players.
- You can visit NPC quest givers named **QuestMaster** or **Adventurer** (Citizens NPCs) to receive quests.
- Higher kill quest tiers send you after dangerous mobs like Ghasts, Pig Zombies, Giants, and Humans — come prepared!
