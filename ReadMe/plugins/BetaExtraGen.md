# BetaExtraGen

## What It Does
BetaExtraGen adds extra world generation features, creating abandoned structures, ruins, mine shafts, ruined portals, and ravines that spawn naturally in newly generated terrain.

## How It Affects You
As you explore new chunks, you may discover:
- **Ruins** — Crumbling stone structures scattered across the landscape.
- **Abandoned Mines** — Tunnel networks underground (up to 40 blocks deep, 4 blocks tall).
- **Full Mines** — Larger, more complete mine systems.
- **Mine Entrances** — Surface-level entrances leading into underground mines.
- **Ruined Portals** — Broken Nether portal frames in the overworld.
- **Ravines** — Deep crevasses (40 blocks long, 30 blocks deep).

## Generation Frequency
Each new chunk has a **50% chance** to roll for structures. If it passes, each structure type rolls independently:

| Structure | Chance |
|-----------|--------|
| Ruins | 2% |
| Mine | 3% |
| Full Mine | 1% |
| Mine Entrance | 1% |
| Ruined Portal | 2% |
| Ravine | 2% |

Some structures can overlap in the same chunk (e.g., ruins + mine, ravine + portal).

## Notes
- No commands — structures generate automatically in new terrain.
- Explore far from spawn to find new structures in freshly generated chunks.
- Abandoned mines can be a great source of resources and adventure!
