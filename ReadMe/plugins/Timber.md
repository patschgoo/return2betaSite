# Timber

## What It Does
Timber makes tree chopping easier by automatically breaking an entire tree when you chop the bottom log, as long as you're using an axe.

## How It Affects You
When you chop a tree trunk block with an axe, the entire tree falls at once. Trees auto-replant saplings.

## Configuration
- **Sneak Mode:** ON — you must be **sneaking (holding Shift)** to activate Timber. Normal chopping works as usual when not sneaking.
- **Max Logs:** Up to **25 log blocks** per tree.
- **Auto-Replant:** Enabled — a sapling is automatically planted where the tree was.
- **All axe types work:** Wood, Stone, Iron, Gold, and Diamond axes.

## How to Use
1. Hold an axe.
2. **Sneak** (hold Shift).
3. Break the bottom log block of a tree.
4. The entire tree (up to 25 logs) breaks and drops.
5. A sapling is automatically planted.

## Tips
- If you want to break just one log block, simply chop without sneaking.
- This saves a lot of time when gathering wood!
