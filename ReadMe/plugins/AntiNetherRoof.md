# AntiNetherRoof

## What It Does
AntiNetherRoof prevents players from accessing the top of the Nether (above the bedrock ceiling), which is normally reachable through glitches.

## How It Affects You
If you attempt to get on top of the Nether bedrock ceiling, you will be prevented from doing so. This closes an exploit that allows players to build or travel in areas not intended for gameplay.

## Notes
- No commands — this works automatically.
- Attempting to glitch above the Nether ceiling will simply not work.
