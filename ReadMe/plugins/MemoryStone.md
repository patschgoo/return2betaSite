# MemoryStone

## What It Does
MemoryStone adds a teleportation system using physical stone structures you build in the world. Build a Memory Stone, memorize it, then teleport between memorized locations using a compass.

## How It Affects You
Members can build and break Memory Stones, and use them for fast travel. You need a **compass** to teleport and must stand near a stone to activate it.

## Stone Types & Costs

### Memory Stone (Local, Same World)
- **Build Cost:** $1,000
- **Memorize Cost:** $200
- **Teleport Cost:** $50 per use
- **Structure:** 3×3 stone base with an obsidian pillar (stone + obsidian column)
- **Limitation:** Same world only

### Crossworld Stone
- **Build Cost:** $3,000
- **Memorize Cost:** $750
- **Teleport Cost:** $75 per use
- **Structure:** 3×3 iron block base with an obsidian pillar
- **Feature:** Travel between worlds (e.g., overworld to nether)

### Global Stone (Everyone Can See)
- **Build Cost:** $2,000
- **Memorize Cost:** $500
- **Teleport Cost:** $65 per use
- **Structure:** 5×5 gold block base with obsidian/diamond pillar
- **Feature:** Automatically memorized for all players

### Global Crossworld Stone
- **Build Cost:** $5,000
- **Memorize Cost:** $1,000
- **Teleport Cost:** $150 per use
- **Structure:** 5×5 gold block base with diamond pillar
- **Feature:** Global + cross-world travel

## How to Use
1. **Build** the stone structure according to the pattern for your chosen type.
2. **Place a sign** on the obsidian pillar to name your stone.
3. **Right-click** the sign to **memorize** the location.
4. To **teleport**, hold a **compass** and right-click the sign, then select your destination.
5. There is a **10-second cooldown** between teleports and a **3-second casting time** (don't move!).

## Commands
| Command | Description |
|---------|-------------|
| Right-click sign with compass | Open teleport destination menu. |
| Right-click sign empty-handed | Memorize the stone location. |

## Notes
- A compass has **50 uses** before it wears out.
- Lightning strikes the stone when it's created, broken, or when you teleport.
- You can also use **Glowstone Dust** for stone-to-stone teleportation (10 uses per dust).
- Locations are sorted by distance from your current position.
- If you can't afford the teleport, the stone will tell you the cost.
