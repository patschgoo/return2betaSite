# PluginReloader

## What It Does
PluginReloader allows staff to reload individual plugins without restarting the entire server. It automatically detects modified plugin files and can reload them.

## How It Affects You
This is a staff-only tool. You may occasionally notice a brief hiccup if a plugin is reloaded while you're playing, but otherwise this has no impact on Members.

## Notes
- No commands for regular players.
- Auto-reload of modified plugins is enabled.
