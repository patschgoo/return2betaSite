# WorldGuard

## What It Does
WorldGuard provides region protection and server-wide gameplay rules. Staff use it to protect spawn and other important areas.

## How It Affects You
WorldGuard runs in the background to protect important server areas. As a member, you benefit from:

- **Fire spread** is disabled server-wide for safety.
- **Lightning fire** is disabled.
- Certain **protected regions** (like spawn) cannot be griefed.
- Creeper block damage is prevented in protected zones.

## Notes
- Members cannot use WorldGuard commands.
- If you need an area protected, use PreciousStones protection blocks instead, or ask staff.
- WorldGuard regions are defined and managed by staff only.
