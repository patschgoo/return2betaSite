# ReciPlus (Custom Recipes)

## What It Does
ReciPlus adds custom crafting recipes to the server, allowing items that normally can't be crafted to be made at a crafting table.

## How It Affects You
You can craft **chainmail armor** using Iron Ingots and String!

## Custom Recipes

### Chainmail Helmet

```crafting:chainmail_helmet
string iron_ingot string
iron_ingot _ iron_ingot
_ _ _
```

### Chainmail Chestplate

```crafting:chainmail_chestplate
string _ string
iron_ingot string iron_ingot
string iron_ingot string
```

### Chainmail Leggings

```crafting:chainmail_leggings
string iron_ingot string
iron_ingot _ iron_ingot
string _ string
```

### Chainmail Boots

```crafting:chainmail_boots
string _ string
iron_ingot _ iron_ingot
_ _ _
```

**Materials:** Iron Ingot and String.

## Notes
- No commands needed — just use a crafting table as normal.
- Chainmail armor provides protection between iron and leather, making it a useful mid-tier option.
