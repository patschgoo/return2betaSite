# WorldEdit

## What It Does
WorldEdit is a powerful in-game map editor used by staff to make large-scale terrain and building modifications.

## How It Affects You
WorldEdit is available to **Admin rank and above only**. Regular members cannot use WorldEdit commands. You may see staff using it to modify terrain or help with server builds.

## Notes
- Members cannot use any WorldEdit commands.
- If you need large-scale building help, ask a staff member.
