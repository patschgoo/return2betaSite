# SkyPirates

## What It Does
SkyPirates turns regular Minecraft boats into flying planes, submarines, hovercrafts, gliders, and drills.

## How It Affects You
Place and enter a boat, then use `/sky <mode>` to transform it. Each mode changes how the boat moves — planes fly, submarines dive, hovercrafts float above land, gliders descend slowly, and drills tunnel through blocks.

## Modes
| Mode | Command | Description |
|------|---------|-------------|
| Plane | `/sky p` | Boat flies through the air |
| Submarine | `/sky s` | Boat dives and moves underwater |
| Hoverboat | `/sky h` | Boat hovers above the ground |
| Glider | `/sky g` | Boat floats gently downward |
| Drill | `/sky d` | Boat tunnels through solid blocks |
| Normal | `/sky` + anything else | Resets to a regular boat |

## Commands
| Command | Description |
|---------|-------------|
| `/sky <mode>` | Switch boat mode (must be in a boat) |
| `/sky help` | List all available modes |

Aliases: `/skypirates`, `/skypi`

## Controls
- **Left-click** while in a boat — attack/special action for current mode.
- **Right-click** while in a boat — secondary action (e.g. throttle, altitude).
- Wearing a **helmet** enables additional interactions.

## Notes
- You must be sitting in a boat to switch modes.
- Switching to a mode outside a boat gives an error message.
- No config files — just get in a boat and go.
