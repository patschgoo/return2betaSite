# DisableCrafts

## What It Does
DisableCrafts can prevent certain items from being crafted. Server admins configure which recipes are blocked.

## How It Affects You
Currently, **no crafting recipes are disabled**. You can craft everything normally, including TNT.

## Notes
- No commands — any restrictions are applied automatically.
- If a recipe is disabled in the future, you simply won't be able to craft that item at a crafting table.
