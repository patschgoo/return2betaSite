# SimpleClans

## What It Does
SimpleClans allows players to form clans, manage members, set rivals and allies, and communicate through clan chat. It adds a social and competitive layer to the server.

## How It Affects You
Members have full member-level clan permissions. You can create clans, join clans, manage your clan, and use clan chat.

## Clan Basics
- **Clan creation costs $100** (tag: 2–5 characters, name: 2–25 characters).
- **Clan verification costs $1,000** — new clans must be verified before full features unlock. Unverified clans are purged after 2 days.
- Inactive clans are purged after **7 days** of inactivity.
- Inactive player data is purged after **30 days**.
- Setting allies or rivals requires a minimum clan size of **3 members**.
- Clan chat tags appear before your name in chat.

## Commands
| Command | Description |
|---------|-------------|
| `/clan create <tag> <name>` | Create a new clan. |
| `/clan invite <player>` | Invite a player to your clan. |
| `/clan accept` | Accept a clan invitation. |
| `/clan deny` | Deny a clan invitation. |
| `/clan leave` | Leave your current clan. |
| `/clan roster` | View your clan's member list. |
| `/clan profile` | View your clan's profile. |
| `/clan stats` | View clan statistics. |
| `/clan rival <tag>` | Set a clan as rival. |
| `/clan ally <tag>` | Request an alliance with a clan. |
| `/clan bb` | View the clan bulletin board. |
| `/clan bb <message>` | Post to the clan bulletin board. |
| `/clan chat <message>` | Send a message in clan chat. |
| `/clan toggle` | Toggle clan chat mode on/off. |
| `/clan lookup <player>` | Look up another player's clan info. |
| `/clan list` | List all clans on the server. |
| `/clan top` | View top clans by KDR. |
| `/clan leaderboard` | View the player leaderboard. |

## Kill Weights
| Player Type | KDR Weight |
|-------------|------------|
| Rival clan member | 1.5x |
| Neutral player | 1.0x |
| Civilian (no clan) | 0.5x |

## Tips
- The clan bulletin board shows on login and holds 10 messages.
- Enable clan capes if using Spout client.
- Clan tags display in chat in gray brackets by default.
