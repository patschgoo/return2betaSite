# HeroicDeath

## What It Does
HeroicDeath replaces the default death messages with humorous and descriptive custom messages. When a player dies, everyone on the server sees a colorful death announcement.

## How It Affects You
This plugin is purely cosmetic — it just makes death messages more entertaining. Player names appear in dark aqua, items in gold, and messages in red.

## Death Messages

| Death Cause | Example Messages |
|-------------|-----------------|
| **Drowning** | "%player drowned", "%player is swimming with the fishes" |
| **Cactus** | "%player poked a cactus, but the cactus poked back" |
| **Fire** | "%player forgot how to stop, drop and roll" |
| **Explosion** | "%player apparently has an explosive personality" |
| **Creeper** | "%player hugged a creeper" |
| **Fall** | "%player took a leap of faith" |
| **PvP** | "%attacker killed %player wielding %item" |
| **Void** | "%player fell into the Gap" |
| **Monsters** | "%player was killed by an angry %monster" |
| **Lava** | "%player became obsidian", "%player took a bath in a lake of fire" |
| **Suffocation** | "%player suffocated" |
| **Dispenser** | "%player got shot by a dispenser" |
| **Lightning** | "%player has an electrifying personality" |
| **Other** | "%player was killed by Herobrine" |

## Notes
- Death events are logged with timestamps.
- There are no commands for this plugin — it works automatically.
