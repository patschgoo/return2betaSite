# BetaSprint

## What It Does
BetaSprint adds a sprinting/running mechanic that lets you move faster by using a special key or item trigger.

## How It Affects You
Sprinting is available to all players. It uses an energy system — you move faster while sprinting but your energy drains.

## How to Sprint
- **Trigger Mode:** Press the sprint **key** to start sprinting (key-based trigger, no item needed).
- **Speed:** 0.6x boost to movement speed.
- **Energy drain:** 2 energy per second while sprinting.
- **Energy regen:** 1 energy per second while not sprinting.
- **Sprinting in air:** Allowed.
- **Sprinting in water:** Allowed.
- There is a **5-tick activation delay** before sprinting starts.
- No permission required — all players can sprint.

## All Trigger Modes
BetaSprint supports these activation modes (only one is active at a time):

| Mode | Description |
|------|-------------|
| **key** | Press the sprint key to activate. **(Active on this server)** |
| **item** | Hold a specific item (e.g., Feather, ID 288) in your hand. |
| **sneak** | Use the sneak key to toggle sprinting. |

## Notes
- Energy status messages appear every 5 seconds while sprinting.
- There is no command needed — just use the sprint key.
- If you run out of energy, you'll stop sprinting until it regenerates.
