# BitTweaks

## What It Does
BitTweaks applies various small fixes and gameplay tweaks to improve the Minecraft experience. These are server-wide and affect all players.

## Active Tweaks on This Server
| Tweak | Description |
|-------|-------------|
| **Fix Water Fall Damage** | Falling into water from a height no longer causes fall damage (vanilla bug fix). |
| **Fix Block Drops** | Blocks drop the correct items when broken (vanilla bug fix). |
| **Fix Pig Saddle** | Pigs properly drop saddles when killed. |
| **Disable Pig Zombie Drops** | Pig Zombies no longer drop items (reduces Nether farming). |
| **Fix Slime Split** | Slimes split correctly when killed. |
| **Fix Cobweb Fall Damage** | Landing in cobwebs no longer causes fall damage (vanilla bug fix). |

## Inactive Tweaks
- Leaf decay is NOT disabled (leaves decay normally).
- TNT behaves normally (not alpha-style).
- Explosions still break blocks.
- Sleeping is enabled.
- Shearing and bonemeal work normally.

## Notes
- No commands — these tweaks are applied automatically.
- These are all quality-of-life fixes that improve the vanilla experience.
