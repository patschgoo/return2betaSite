# GetStairs

## What It Does
GetStairs improves stair block drops so that when you break stair blocks, you actually get the stair item back instead of the base material.

## How It Affects You
Breaking stair blocks drops the stair block itself, not the raw material. This makes it easy to reuse stairs in your builds.

## Notes
- No commands — this works automatically.
- Applies to all stair types.
