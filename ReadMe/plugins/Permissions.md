# Permissions (Legacy)

## What It Does
This is the legacy Bukkit Permissions plugin that provides basic permission support. On this server, **PermissionsEx** is the active permissions manager and has replaced this plugin.

## How It Affects You
This plugin has no direct impact — PermissionsEx handles all permission management. See the PermissionsEx readme for rank details.

## Notes
- No commands for regular players.
