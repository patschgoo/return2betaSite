# CustomPortalSize

## What It Does
CustomPortalSize allows Nether portals to be built in non-standard sizes, both smaller and larger than the vanilla 4×5 portal.

## How It Affects You
You can build Nether portals in custom dimensions:
- **Minimum width:** 1 block (smaller than vanilla!)
- **Maximum width:** 23 blocks
- **Minimum height:** 2 blocks
- **Maximum height:** 23 blocks

## Notes
- No commands needed — just build a portal frame with obsidian in your desired size and light it with flint and steel.
- The standard vanilla portal size (2×3 interior) still works.
- This lets you build grand portal archways or compact 1-wide portals.
