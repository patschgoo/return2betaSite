# InventorySort

## What It Does
InventorySort lets you sort your inventory and chests with a simple toggle command.

## How It Affects You
Members can toggle automatic inventory stacking and use the sorting wand on chests.

## Features
- **Stack Toggle:** Enable/disable automatic stacking of items in your inventory.
- **Chest Sorting Wand:** Right-click a chest with a **Stick** (ID 280) to sort its contents.

## Commands
| Command | Description |
|---------|-------------|
| `/isort` | Toggle automatic inventory stacking on/off. |
| `/isort stack` | Manually stack items in your inventory. |

## Notes
- Automatic stacking is off by default — you must toggle it on.
- The sorting wand works on any chest you have access to.
