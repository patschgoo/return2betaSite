# EssentialsChat

## What It Does
EssentialsChat handles the chat formatting on the server. It works alongside Essentials to format player messages with rank prefixes and colors.

## How It Affects You
- Chat is **global** — everyone on the server sees your messages regardless of distance.
- Your chat format shows as: `<[Rank] PlayerName> message`
- Your rank prefix (e.g., `[Member]`) appears before your name in green.

## Notes
- No special commands — just type in chat normally.
- Use `/msg <player> <message>` for private messages (see Essentials).
