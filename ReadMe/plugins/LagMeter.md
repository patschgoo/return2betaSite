# LagMeter

## What It Does
LagMeter monitors server performance (TPS — ticks per second) and provides a command to check current server lag.

## How It Affects You
Members can check the server's current performance at any time.

## Commands
| Command | Description |
|---------|-------------|
| `/lag` | Check the current server TPS and lag status. |

## Notes
- A healthy server runs at 20 TPS. Lower numbers mean more lag.
- The displayed value is an average over 10 readings for accuracy.
- If the server is consistently laggy, consider starting a vote to restart (see Vote plugin).
