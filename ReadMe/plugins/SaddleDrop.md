# SaddleDrop

## What It Does
SaddleDrop makes pigs always drop their saddle when killed, so you can recover saddles instead of losing them permanently.

## How It Affects You
Kill a saddled pig and the saddle will drop as an item, allowing you to reuse it.

## Notes
- No commands — this works automatically.
- Saddles are rare items, so this is a useful quality-of-life fix.
