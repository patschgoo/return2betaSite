# SuperLog

## What It Does
SuperLog records various server events including block changes, chest interactions, and commands for staff to review in case of griefing or disputes.

## How It Affects You
SuperLog runs silently in the background. Your actions (block placement, block breaking, chest access, commands) may be logged. This helps staff investigate griefing and resolve disputes.

## Notes
- No commands for regular players.
- If you've been griefed, staff can use the logs to find out who did it and when.
- This is why you should always report griefing — the evidence is recorded automatically.
