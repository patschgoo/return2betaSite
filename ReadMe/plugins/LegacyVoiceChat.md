# LegacyVoiceChat

## What It Does
LegacyVoiceChat adds proximity-based voice chat to the server, allowing nearby players to communicate using voice instead of text.

## How It Affects You
If you have a compatible client-side mod installed, you can talk to players within **48 blocks** of you using voice chat. Players further away won't hear you.

## How to Connect
- Connect to the voice server on **port 25566** (same server IP, different port).
- The voice server runs alongside the game server.

## Default Button Layout
The default keybind to talk is typically **V** (push-to-talk). This depends on your client-side mod configuration.

### Changing Keybinds
Keybinds are configured in the **client-side mod**, not on the server. To change them:
1. Open your Minecraft client settings or the voice chat mod's settings menu.
2. Look for the voice chat keybind options.
3. Reassign the push-to-talk key or other voice controls as desired.

## Server Configuration
- **Voice port:** 25566
- **Max distance:** 48 blocks (proximity range)

## Notes
- Requires a compatible client-side mod to use voice chat.
- Voice chat is proximity-based — only nearby players can hear you.
- The server cannot control client keybinds — those are set in your mod options.
