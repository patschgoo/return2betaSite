# SimpleSignEdit

## What It Does
SimpleSignEdit lets you edit text on signs that are already placed in the world without having to break and replace them.

## How It Affects You
You can use the editing tools to change sign text. The plugin supports two interaction items:
- **Stick** (ID 280) — View who placed a sign.
- **Feather** (ID 288) — Edit a sign's text (set mode).

## How to Use
- Hold the appropriate item and right-click a sign.
- When editing, you'll be prompted to enter new text for each line.
- Both sneaking and standing modes work for interactions.

## Notes
- Signs that are break-protected cannot be broken, but editing is not restricted to Members.
- Auto-save is enabled — changes are saved immediately.
