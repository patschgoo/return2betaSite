# Vault

## What It Does
Vault is a backend plugin that connects the economy (iConomy), permissions (PermissionsEx), and chat systems together. It allows other plugins to interact with the economy and permissions without needing direct integration.

## How It Affects You
Vault runs entirely in the background. You won't interact with it directly, but it's the reason plugins like LWC, PreciousStones, and SimpleClans can charge you money and check your permissions.

## Notes
- No commands for regular players.
- If economy features aren't working, it's usually a Vault connection issue — notify staff.
