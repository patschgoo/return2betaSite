# BetaServerCommands

## What It Does
BetaServerCommands provides server management utilities including an automatic restart system.

## How It Affects You
If the server restarts (e.g., after a successful restart vote), this plugin handles the restart process. You may be kicked briefly while the server restarts.

## Notes
- No commands for regular players.
- The server restart process is automatic and handled by staff or the vote system.
