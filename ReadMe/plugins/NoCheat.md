# NoCheat

## What It Does
NoCheat is an anti-cheat plugin that monitors player behavior and prevents cheating such as speed hacking, fly hacking, and illegal block interactions.

## How It Affects You
NoCheat runs silently in the background. If you play fairly, you'll never notice it. It monitors:
- **Movement:** Detects speed hacks, fly hacks, and no-fall-damage cheats.
- **Block breaking:** Prevents impossible block break speeds or reaching blocks too far away.
- **Block placing:** Prevents placing blocks at impossible distances or speeds.
- **Interactions:** Prevents spam-clicking and impossible interactions.
- **Chat:** Monitors for spam and command spam.

## Notes
- No commands for regular players.
- If you experience a false positive (getting teleported back or blocked from an action you should be able to do), notify staff using `/report` or `/helpop`.
- Playing normally and not using any hacked clients means you'll never have issues with this plugin.
