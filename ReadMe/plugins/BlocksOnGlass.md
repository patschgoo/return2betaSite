# BlocksOnGlass

## What It Does
BlocksOnGlass allows you to place blocks on top of glass blocks, which vanilla Minecraft doesn't normally allow.

## How It Affects You
You can build on glass — place any block on top of glass blocks just like you would on any other solid block.

## Notes
- No commands — this works automatically.
- Great for building skylights, glass floors with decorations on top, and modern builds.
