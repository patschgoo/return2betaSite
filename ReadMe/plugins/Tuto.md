# Tuto (Tutorials)

## What It Does
Tuto provides an in-game tutorial system. Tutorials are written by staff and cover various topics about the server's features and mechanics.

## How It Affects You
Members (and Visitors) can list and read available tutorials to learn about the server.

## Commands
| Command | Description |
|---------|-------------|
| `/tutolist` | List all available tutorials (5 per page). |
| `/tutolist <page>` | View a specific page of the tutorial list. |
| `/tuto <name>` | Open a specific tutorial. |
| `/tuto <name> <page>` | Jump to a specific page in a tutorial. |
