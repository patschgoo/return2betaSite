# MiningTNT

## What It Does
MiningTNT modifies TNT behavior to only destroy specific block types, making it useful for mining rather than griefing.

## How It Affects You
TNT explosions will only destroy specific blocks (Stone, Sand, Gravel, Leaves, Glass, Sandstone) instead of everything. TNT will not work above Y=70 (below sea level only).

## Configuration
- **Affected blocks:** Stone (1), Sand (12), Gravel (13), Leaves (18), Glass (20), Sandstone (24)
- **Max altitude:** Y=70 — TNT has no effect above this height.
- **Creeper nerf:** Disabled (creepers explode normally).
- **Creepers act like TNT:** No (creepers destroy all blocks as usual).

## Notes
- No commands — this modifies TNT behavior automatically.
- TNT is mainly useful for underground mining on this server.
- You can craft TNT normally (5 gunpowder + 4 sand).
