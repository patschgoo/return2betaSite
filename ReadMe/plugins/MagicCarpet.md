# MagicCarpet

## What It Does
MagicCarpet creates a floating glass platform beneath you that you can fly around on, like a magic carpet.

## How It Affects You
MagicCarpet is **not available to Member rank** players. It requires Moderator or higher rank permissions. You may see staff flying around on glass platforms — that's this plugin in action.

## Notes
- Members cannot use Magic Carpet.
- If you see floating glass platforms, they belong to staff members using this plugin.
