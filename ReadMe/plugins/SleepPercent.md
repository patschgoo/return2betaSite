# SleepPercent

## What It Does
SleepPercent changes the sleeping mechanic so that only a percentage of online players need to sleep in beds to skip the night, rather than requiring everyone.

## How It Affects You
Not all players need to be in a bed to skip the night. When enough players sleep, the night is skipped for everyone.

## Notes
- No commands — just use a bed as normal.
- Great for servers where some players want to keep playing at night while others want to sleep through it.
