# Sprint

## What It Does
Sprint is a sprinting plugin that lets you move faster using an energy-based system. The server also has **BetaSprint** installed (see below).

## Sprint Activation Modes
Sprint plugins support several activation methods. The server can use one at a time:

| Mode | How It Works |
|------|-------------|
| **Key-based** | Press the sprint key to start sprinting. No item needed. |
| **Item held** | Hold a specific item in your hand to sprint. |
| **Item required** | Have a specific item in your inventory (doesn't need to be held). |
| **Command-based** | Type `/sprint` to toggle sprinting on/off. |

## Active Configuration on This Server
The server uses **BetaSprint** with **key-based** trigger mode. Press the sprint key to run faster.

### BetaSprint Settings (Active)
- **Speed boost:** 0.6x faster
- **Energy drain:** 2 energy per second while sprinting
- **Energy regen:** 1 energy per second while standing still
- **Activation delay:** 5 ticks
- **Sprinting in air:** Allowed
- **Sprinting in water:** Allowed
- **Item required:** None (key-based mode)

### Sprint Plugin Settings (Also Installed)
- **Speed boost:** 0.6x faster
- **Energy drain:** 1 energy per second
- **Energy regen:** 1 energy per second
- **High jump:** Disabled
- **Item/command requirement:** None

## Notes
- Energy status messages appear every 5 seconds while sprinting.
- If you run out of energy, you'll stop sprinting until it regenerates.
- No commands needed — just use the sprint key.
