# MobSpawn

## What It Does
MobSpawn controls which mobs can naturally spawn in the world.

## How It Affects You
All mob types are currently **enabled** on this server:

### Hostile Mobs
- Zombie ✔
- Skeleton ✔
- Spider ✔
- Creeper ✔
- Ghast ✔
- Slime ✔
- Pig Zombie ✔

### Passive Mobs
- Pig ✔
- Cow ✔
- Sheep ✔
- Chicken ✔
- Squid ✔
- Wolf ✔

## Notes
- No commands — mob spawning is controlled server-side.
- All vanilla mobs spawn normally. The server has not disabled any mob types.
