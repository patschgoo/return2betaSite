# JWildTP (Wild Teleport)

## What It Does
JWildTP teleports you to a random location in the wilderness, perfect for finding new areas to explore or settle.

## How It Affects You
Members can use the wild teleport command to find a random spot in the overworld.

## Configuration
- **World:** Overworld only (Nether teleportation is disabled).
- **Random radius:** Between 10 and 15 blocks from world center (0, 0).
- **Cooldown:** None — you can use it as often as you like.

## Commands
| Command | Description |
|---------|-------------|
| `/wild` | Teleport to a random location in the wilderness. |

## Notes
- The teleport destination is random within the configured radius.
- Nether wild teleport is disabled.
- Great for finding unclaimed land for your base!
