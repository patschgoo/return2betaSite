# EssentialsProtect

## What It Does
EssentialsProtect provides server-wide block and entity protection rules. It prevents certain griefing methods and controls environmental damage.

## How It Affects You
The following protections are active server-wide:
- **Fire spread** is disabled — fire won't spread to wooden structures.
- **Lava fire spread** is disabled.
- **Lightning fire** is disabled.
- Creeper explosions, TNT, and fireballs still work normally but won't cause fire.
- Portal creation is allowed.
- PvP is enabled (players with PvP permission can fight each other).
- Fall damage, drowning, lava damage, and other environmental damage work normally.

## Notes
- No commands for regular players.
- These protections help keep the server safe from accidental destruction.
