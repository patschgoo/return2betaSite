# Command iConomy

## What It Does
Command iConomy charges money for using certain commands, integrating the economy with command usage.

## How It Affects You
Some commands cost money to use. The fee is deducted from your balance automatically.

## Command Costs
| Command | Cost |
|---------|------|
| `/cprivate` | $100 (lock a chest/door as private) |
| `/cpassword` | $75 (lock a chest/door with a password) |
| `/cpublic` | $25 (lock a chest/door as public) |

## Notes
- If you don't have enough money, you'll see: *"Insufficient funds. [cost] needed."*
- The cost is deducted immediately upon using the command.
- Check your balance with `/money` or `/balance` before using paid commands.
