Server Worlds

Three worlds, one server — pick your play style

Server World

The full Return2Beta experience — all plugins, all features

Overview

The Server World is the main world and default spawn. This is where everything happens: economy, land protection, warps, homes, chat features, and every quality-of-life enhancement the server offers. All plugins are active and all commands are available.

What’s Available

  • Full economy — earn and spend with iConomy, shops via Showcase
  • Land protection — protect your builds with PreciousStones and LWC chests
  • Homes & warps — set homes, create warps, teleport to players
  • Chat features — colored chat, Discord bridge, web chat
  • Quality-of-life plugins — Timber, Stackable, InventorySort, Pickup, and more
  • Quests & clans — QuestPlus objectives and SimpleClans groups
  • Redstone & rails — WirelessRedstone, FalseBook, TrainCarts
  • Live map — Dynmap with mob tracking
  • Voting rewards — Vote plugin + RankUp system

Active Plugins

All 80+ plugins are active in this world, including Essentials, iConomy, LWC, PreciousStones, WorldGuard, Dynmap, HeroicDeath, Timber, TrainCarts, and many more. Visit the Plugins page for the full list and documentation.

Nether

Nether portals work normally with vanilla behavior. Build a portal, light it, and travel to the Nether as usual.

This is the default world — you spawn here when joining the server.

Creative World

Build freely — item spawning, WorldEdit, flight, instant breaking

Overview

The Creative World is a building paradise adjusted to work like creative mode. When you enter, you get creative flight, instant block breaking, item spawning, and access to WorldEdit tools. Build whatever you can imagine without gathering a single resource.

How to Enter

  • Type /dd creative from the Server World
  • Or right-click a [CD] portal sign
  • Type /dd leave to return to the Server World

What You Can Do

  • Spawn items — use /i <item> or /give to get any block or item
  • Creative flight — fly freely, enabled automatically on enter
  • Instant breaking — break any block instantly with Instabuild
  • WorldEdit tools — selection, region fill, clipboard, undo/redo, shapes
  • Navigation — /back, /top, /up, /jump for quick movement
  • Co-building — /tpa, /tpahere to invite friends to your build
  • Utilities — /getpos, /clearinventory, /near, /me

What’s Blocked

Dangerous commands are blocked to keep the server safe. You cannot use:

  • Server management — /stop, /reload, /op, /deop, /whitelist
  • Permissions — /pex, /permissions, /rank, /rankup
  • Moderation — /ban, /kick, /mute, /jail
  • Economy — /eco, /money, /pay, /balance
  • Region protection — /wg, /region, /lwc, /cprivate
  • Other dangerous — /sudo, /god, /enchant, /vanish

Nether

Nether portals are disabled in the Creative World. You cannot light or enter nether portals here.

Important Notes

  • Your inventory is completely separate from the Server World and Survival World
  • Your position is saved — leave and return later to the exact same spot
  • Creative permissions are granted on entry and revoked on exit automatically
  • Chat messages show a yellow [CD] tag

Survival World

Raw, old-school survival — no plugins, no commands, nothing

Overview

The Survival World is a pure, unfiltered survival experience inspired by servers like 2b2t. There are no plugins, no commands, no teleports, no homes — nothing. Just you, the terrain, and whatever you can mine and craft. Not even server operators can run commands inside.

How to Enter

  • Type /dd survival from the Server World
  • Or right-click a [DD] portal sign
  • Type /dd leave — the only command that works inside

The Rules

  • Every command is blocked — no /home, /tp, /gamemode, /give, nothing
  • Being OP does not bypass the block
  • The only commands that work: /dd leave and /dd help
  • No plugins of any kind — no protection, no economy, no kits
  • Pure survival — mine, craft, build, fight, explore

Commands

CommandDescription
/dd leaveReturn to the Server World
/dd helpShow available commands

That’s it. Those are the only two commands.

Nether

The Survival World has its own separate nether. Nether portals work normally — build a portal and light it to travel. The nether is private to this world (not shared with the Server World). Command blocking still applies in the nether. Dying in the nether respawns you at the Survival World overworld spawn.

Important Notes

  • Your inventory is completely separate from the Server World and Creative World
  • Your position is saved — leave and return later to the exact same spot
  • If you leave from the nether and return later, you’ll be placed back at your nether position
  • Chat messages show a purple [DD] tag
  • Inventory carries between the overworld and nether (same survival experience)

This world is for players who want the raw, unassisted Minecraft experience. No safety nets.
